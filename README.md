💡 Task 2: Sensor-Based Simulation (LDR Controlled LED)
📖 Description

In this task, a simulation was created using Tinkercad where an LED automatically turns ON and OFF based on light intensity using an LDR (Light Dependent Resistor).

⚙️ Components Used

Arduino UNO

LDR (Light Sensor)

LED

Resistors

Breadboard

🔄 Working Principle

Low light (dark) → LED turns ON

High light (bright) → LED turns OFF

🎯 Outcome

This task demonstrates how sensors can be used in IoT systems for automation, such as smart lighting.

🚦 Task 3: Case Study – IoT in Smart Cities
📖 Description

A detailed case study on the future of IoT in smart cities, including applications like traffic management, energy systems, and public safety.

🔧 Practical Example Included

A traffic light system using Arduino UNO was used to demonstrate smart transportation.

🔄 Working of Traffic Light System

Red LED → Stop

Yellow LED → Wait

Green LED → Go

Operates in a timed sequence

🌐 IoT Concept

In real smart cities:

Sensors detect traffic density

Data is processed in cloud systems

Signals are adjusted dynamically

🎯 Outcome

Understanding how IoT can improve urban infrastructure and traffic management.
